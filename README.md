# Ferretería App

## Descripción
Una aplicación web para una ferretería que permite mostrar productos y administrar el inventario.

## Características
- Visualización de productos para clientes
- Panel de administración con autenticación
- CRUD de productos

## Requisitos Previos
- Node.js
- MySQL

## Configuración

### Backend
1. Navegar al directorio backend
2. Instalar dependencias: `npm install`
3. Configurar `.env` con tus credenciales de base de datos
4. Crear base de datos MySQL llamada `ferreteria`
5. Iniciar servidor: `npm run dev`

### Frontend
- Abrir `index.html` en un navegador
